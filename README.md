# Planning Poker

A mobile-first Planning Poker application built with Vite, Svelte, and TailwindCSS.

## Features

- 13 estimation cards: 0, 0.5, 1, 2, 3, 5, 8, 13, 20, 40, 100, ?, ☕️
- Mobile-responsive design
- Simple and intuitive interface
- Large card display for easy visibility
- Standalone application that can be used from any phone

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This app is configured to be deployed to GitHub Pages. It will automatically deploy on pushes to the main branch.
